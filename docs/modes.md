# Modes

## Safe

Chat with Gemini, no shell-like helpers.

Use this if you are cautious, tired, or not in the mood to debug a self-inflicted mess.

## Balanced

The default.

You get normal project navigation helpers like `/project`, `/ls`, and `/git`, but still under the workspace sandbox.

## Power

Admin only.

This is where you stop pretending you want guard rails.

Use it only if you understand what the host can do and you are willing to own the result.
