- `/help`
- `/status`
- `/new`
- `/continue`
- `/end`
- `/export [md|json|html]`

## Project navigation

- `/project <path>`
- `/repo <path>`
- `/projects`
- `/pwd`
- `/ls [path]`

## Power tools

- `/mode [safe|balanced|power]`
- `/git`
- `/actions`

## Quick note

If you are just trying to vibecode and not become a part-time infra archaeologist:

1. use `/project`
2. send a plain-English prompt
3. use `/new` when the session gets weird
4. run `gemini-telegram check` on the host if the bot starts acting dead
