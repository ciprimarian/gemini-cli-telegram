# Roadmap

## Near term

- richer session actions
- better upload context for code and screenshots
- more tests around subprocess failures
- better admin-facing diagnostics

## Mid term

- optional webhook mode
- optional speech-to-text integration
- more export formats if they stay stable and maintainable
- packaging polish for broader installs

## Not by default

- silent remote command execution with no audit trail
- fancy UI for the sake of it
- pretending Gemini CLI host auth can be replaced by this bridge
